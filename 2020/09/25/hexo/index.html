hexo

发表于 2020-09-25
本文字数： 523 阅读时长 ≈ 1 分钟

1全局安装 hexo-
npm install -g hexo-cli
2进入出案件的目录
hexo init
3进入 gitlab 创建仓库
格式：username.github.io

4安装hexo-deployer-git自动部署发布工具
$ npm install hexo-deployer-git –save

5 hexo clean 清除缓存
6 生成 部署 hexo generate hexo deploy
hexo clean && hexo g && hexo d
