rabbitmq

1、安装依赖

　yum -y install gcc glibc-devel make ncurses-devel openssl-devel xmlto perl wget gtk2-devel binutils-devel

2、下载 erlang

wget http://erlang.org/download/otp_src_22.0.tar.gz

3、解压

4、创建即将安装的目录：mkdir ../erlang

5、配置安装路径：./configure –prefix=/usr/local/erlang (需要在otp_src_22.0目录下)

6、安装 make install

7、环境变量 vi /etc/profile ；

export ERLANG_HOME=/usr/erlang/bin

export PATH=${ERLANG_HOME}/bin:${PATH}

source /etc/profile

8、 安装rabbitmq

9、tar.xz 格式所以需要用到xz，没有需要先安装 yum install -y xz

/usr/bin/xz -d rabbitmq-server-generic-unix-3.7.15.tar.xz

10、 vi /etc/profile ；

export RABBITMQ_HOME=/usr/rabbitmq/rabbitmq3/sbin

export PATH=${RABBITMQ_HOME}/sbin:${PATH}

source /etc/profile

11、创建配置文件目录

可以通过 find / -name rabbitmq-defaults 查找rabbitmq-defaults文件，查看config文件的存储路径

　　mkdir /etc/rabbitmq

12、进入 /usr/local/rabbitmq/sbin 启动rabbitmq： ./rabbitmq-server

13、开启UI管理界面：rabbitmq-plugins enable rabbitmq_management 默认端口：15672

14、 用户管理：

查看所有用户：rabbitmqctl list_users

添加新用户：rabbitmqctl add_user ych 123456

配置权限：rabbitmqctl set_permissions -p “/“ ych “.*” “.*” “.*”

查看用户权限：rabbitmqctl list_user_permissions ych

设置tag：rabbitmqctl set_user_tags ych administrator

rabbitmq-server -detached #后台启动

netstat -ntulp 查看进程信息

mysql

1、检查是否自带mariadb数据库

rpm -qa|grep mariadb

2、删除mariadb数据库

rpm -e –nodeps mariadb-libs-5.5.56-2.el7.x86_64

创建data目录，作为数据库存储位置

groupadd mysql //创建mysql 用户组

useradd -g mysql mysql //创建一个用户名为mysql的用户并加入mysql用户组

chown –R mysql . //把当前目录下的文件及目录的属性改为mysql用户

chgrp –R mysql . //把当前目录下的文件所属的组件改为mysql组

my.cnf配置文件内容：

[mysqld]
bind-address=0.0.0.0
port=3306
user=mysql
basedir=/usr/mysql/mysql
datadir=/usr/mysql/mysql/data
socket=/tmp/mysql.sock
log-error=/usr/mysql/mysql/data/mysql.err
pid-file=/usr/mysql/mysql/data/mysql.pid
#character config
character_set_server=utf8mb4
symbolic-links=0
explicit_defaults_for_timestamp=true

初始化mysql：

./mysqld –defaults-file=/etc/my.cnf –basedir=/usr/mysql/mysql –datadir=/usr/mysql/mysql/data –user=mysql –initialize

先将mysql.server放置到/etc/init.d/mysql中

cp /usr/mysql/mysql/support-files/mysql.server /etc/init.d/mysql

启动mysql服务

service mysqld start

开机自启动：

vim /etc/rc.d/rc.local
添加 “sudo /etc/init.d/mysql start”，

nginx

1、下载安装目录 http://nginx.org/download/

2、进入解压目录 安装到指定位置 ./configure –prefix=/usr/local/nginx

3、make install （make install是把这些编译出来的可执行文件和库文件复制到合适的地方）

4、 参数 -c 指定了配置文件的路径，如果不加的话就是使用默认的配置文件

nginx -c /usr/local/nginx/conf/nginx.conf

5、查询nginx主进程号 ps -ef | grep nginx

6、从容停止Nginx： kill -QUIT 主进程号；

快速停止Nginx ：  kill -TERM 主进程号 ；

强制停止Nginx ： pkill -9 nginx

7、判断Nginx配置是否正确：nginx -t -c /usr/nginx/conf/nginx.conf 或者 /usr/nginx/sbin/nginx -t

8、重启：/usr/nginx/sbin/nginx -s reload

redis

1.安装gcc redis是c语言编写的

yum install gcc-c++

wget http://download.redis.io/releases/redis-5.0.4.tar.gz

5、进入redis目录编译 make

6、安装 make PREFIX=/usr/local/redis install

7.拷贝redis.conf到安装目录 cp redis.conf /usr/local/redis.conf

9.编辑redis.conf
vi redis.conf

1.后台启动，daemonize yes

2.绑定端口，port 6379 默认是6379 需要安全组开放端口

3.绑定IP，bind 192.168.2.128 绑定哪个哪个可以访问 0.0.0.0 为全部ip可访问

4.指定数据存放路径，dir /usr/local/redis/log rdb存放的路径

5.指定持久化方式，appendonly yes

6.requirepass redis129 设置密码

后端启动redis： ./bin/redis-server ./redis.conf

ps aux | grep redis
